Kutuphane Yonetim Sistemi

Bu proje C# ile gelistirilmis basit bir kutuphane yonetim sistemidir.

Projede kullanici kitap ekleyebilir, listeleyebilir, guncelleyebilir, silebilir, odunc alabilir ve iade edebilir.

Projede Kullanilanlar

- C#
- .NET Console Application
- Dosya islemleri

OOP Kullanimi

Projede encapsulation, inheritance ve polymorphism kullanilmistir.

Kisi sinifindan Ogrenci sinifi kalitim almaktadir.

Kitap bilgileri kitaplar.txt dosyasinda saklanmaktadir.

Siniflar

- Kisi
- Ogrenci
- Kitap
- KitapDosya
- Kutuphane
- Program

Calistirma

Projeyi calistirmak icin:

dotnet run

Sonuc

Bu proje ile C# dilinde OOP, CRUD islemleri, dosya kaydetme ve kullanici etkilesimi uygulanmistir.
